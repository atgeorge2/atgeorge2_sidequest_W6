## Description

Simple 2D platformer

---

## Setup and Interaction Instructions

Use arrow keys or WASD to move. Use W or the up arrow to jump. Use the spacebar to attack. Collect the puppies to win. Don't touch the fire or skeletons, or you will lose a heart. If you lose three hearts, you die.
